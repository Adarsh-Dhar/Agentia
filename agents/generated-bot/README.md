# Flash Loan Arbitrageur Bot
> Generated by Meta-Agent | 2026-03-25T02:24:22.286Z

## Architecture
```
DexScreener → price gap detected
     ↓
Rugcheck   → token safety validated  
     ↓
Profit Calc → net profit > $100
     ↓
Aave V3    → flash loan $50000 (0.09% fee)
     ↓
1inch/Jupiter → swap A → swap B
     ↓
Aave V3    → repay loan + fee
     ↓
Profit     → stays in your wallet 🎉
```

## Quick Start
```bash
cp .env.template .env
# Edit .env with your settings

pnpm install
pnpm run build

# Test without real money first:
pnpm run dry-run

# Go live (only after thorough testing):
pnpm start
```
